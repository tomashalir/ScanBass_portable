SCANBASS — EXE + INSTALLER KIT (NO PYTHON NEEDED)
--------------------------------------------------
1) Put this kit at your project root (same level as src\ and ScanBass_portable\).
2) Run: build\pyinstaller\build_scanbass.bat  → dist\ScanBass\ScanBass.exe
3) Run: installer\build_installer.cmd         → installer\Output\ScanBassSetup.exe
Tip: Ensure your working venv is .venv311 (or edit build_scanbass.bat).
